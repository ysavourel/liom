Prototype for the Localization Interchange Object Model API.

  • This document is UNDER DEVELOPMENT and NOT COMPLETE YET and it covers only A PORTION of the object model.
  • This document Is ONLY A PRELIMINARY PROTOTYPE and it HAS NO OFFICIAL STANDING of any kind.

Overview Chart

Legend:  1 : one
         ? : zero or one
         + : one or more
         * : zero, one or more
         # : right-side also implements left-side
       ... : recursive

The object model from the document level down to the unit is as follows:
(Note that the order of the objects in the charts is not significant and not related to the order in any serialization).

IDocument
 |
 +-- ISubDocument+
      |
      +-- IWithContext
      |
      +-- IWithNCObjects
      |    |
      |    +-- INCObjects
      |
      +-- IWithNotes
      |    |
      |    +-- INote*
      |
      +-- ISkeleton?
      |
      +-- (IGroupOrUnit#IGroup*, IGroupOrUnit#IUnit+)
            |            |                     |
            |            |                     +-- See details below
            |            |
            |            +-- (IGroupOrUnit#IGroup*... | IGroupOrUnit#IUnit)*
            |
            +-- IWithContext
            |
            +-- IWithNotes
                 |
                 +-- INote*
